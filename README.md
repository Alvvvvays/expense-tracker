# expense-tracker

Personal expense tracker with charts and monthly reports, built with Vue.js

## Features

- Clean and intuitive user interface
- Responsive design for mobile and desktop
- Well-documented codebase
- Easy to set up and customize

## Getting Started

```bash
git clone https://github.com/Alvvvvays/expense-tracker.git
cd expense-tracker
npm install   # or pip install -r requirements.txt
npm start
```
